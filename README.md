# BeaconAP — Relationship Intelligence

AI-powered relationship intelligence platform for BeaconAP. Helps capital markets professionals stay top of mind with their network at scale.

## Overview

Built as a consulting project for BeaconAP (Babson STR-3501, Spring 2026). The platform captures insights from client conversations across multiple channels and surfaces relevant content matched to each contact's interests.

## Features

- **Dashboard** — Priority actions, network stats, AI-curated content to share
- **Contacts** — Full network with engagement scores, status tags, source tracking
- **Contact Detail** — Unified interaction timeline across all channels, AI-drafted outreach, interest tracking, relationship scoring
- **Pipeline** — Kanban-style relationship funnel (Cold → Nurture → Warm → Hot)
- **Content Feed** — AI-matched articles, podcasts, events, and your own content
- **AI Insights** — Conversion signals, relationship decay warnings, network gaps, content performance
- **Add Contact** — Manual entry + sync from LinkedIn, Contacts, WhatsApp, Instagram

## Connected Sources

- LinkedIn (activity, connections, events)
- Gmail (email tracking, opens, forwards)
- iMessage (conversation signals)
- Granola (meeting notes, call summaries)
- Apollo.io (outreach sequences)

## Tech Stack (Current: Prototype)

Single-file HTML/CSS/JS prototype for demo purposes.

## Tech Stack (Planned: Production)

- **Frontend:** React + TypeScript + Vite
- **Backend:** Supabase (auth, database, realtime)
- **Styling:** Tailwind CSS + shadcn/ui
- **AI:** Claude API (content matching, outreach drafting, interest extraction)
- **Integrations:** LinkedIn API, Gmail API, Granola API, Apollo.io API
- **Deployment:** Vercel
